PDI
===

Parameter Design and Input Method for HPC/PF system.

----

# Requires
  python 2.x (not python 3.x !)
  wxPython 2.8 / 3.0 unicode

# Installation
  nothing to do. you can place the PDI directory as you like.

# Run
  - Linux / MacOSX
    $ PDI/bin/pdi
  - MacOSX with wxPython 2.8 (wxCarbon)
    $ PDI/bin/pdi32
  - Windows
    $ PDI/bin/pdi.bat
